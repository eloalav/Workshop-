Sobre o Workshop

Bem-vindo ao Workshop de Desenvolvimento Web! Neste workshop, você aprenderá sobre os conceitos fundamentais de desenvolvimento web, incluindo HTML, CSS e Flask. Este evento é ideal para iniciantes que desejam obter uma compreensão prática das tecnologias web modernas.

Objetivos do Workshop

  • Compreender a estrutura básica de páginas web usando HTML.
  • Aprender a estilizar páginas web com CSS.
  • Introduzir conceitos de desenvolvimento back-end com Flask.
  • Desenvolver um projeto prático utilizando as tecnologias aprendidas.

Perfil do Público-Alvo

O workshop é direcionado a:

  • Iniciantes em desenvolvimento web que desejam aprender desde o básico.
  • Estudantes e profissionais que buscam aprimorar suas habilidades em tecnologias web.
  • Qualquer pessoa interessada em iniciar uma carreira em desenvolvimento web.
